# x-ui-front
x-ui frontend
Xray admin panel is ready! Here's what I made:
Pages:

Dashboard — General statistics (users, traffic, inbounds) + Weekly traffic graph + Expiring users table
Inbounds — Full list with protocol, port, number of users, traffic, and QR / edit / delete buttons
Users — Show traffic progress bar, IP status, expiration date
Xray status — Version, CPU, RAM, uptime (CPU changes live)
Settings — Panel settings

Features:

Inbound add form with VLESS, VMESS, Trojan, Shadowsocks, Hysteria2, WireGuard protocols
Colored badges for different statuses
Server online animation
Full Persian interface
